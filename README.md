## üöÄ Getting Started

### Prerequisites

- **Python 3.13.9**
- **yt-dlp:** Must be installed and added to your System PATH.
- **FFmpeg:** Required for MP3 conversion and metadata embedding.

### ‚ùì Want to run the tool without .exe?

Follow these steps if you **donot have Python/GIT installed**:
1. Find the `<> Code` Button at the top and click on it.
2. Click on `Download ZIP` from the options that appear.
3. This will download the project files into your download path.
4. Unzip the downloaded project file, and launch the file named as `yt-dlp-toolkit.bat`.


Follow these steps if you have Python installed:
1. Open CMD in your desired path on your System.
2. Clone the Repository:
```
git clone https://github.com/ChaitanyaKumarS2403/yt-dlp-toolkit.git
```
3. Change the directory:
```
cd yt-dlp-toolkit
```
4. Run the following python command to launch the application via python:
```
python yt-dlp-toolkit
```
